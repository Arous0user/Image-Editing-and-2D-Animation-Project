Topic 1: Introduction to Image Editing

Background

Raster vs. Vector Graphics

Raster graphics are composed of a grid of pixels. When you zoom in, you can see the individual pixels. Common raster formats include JPEG and PNG. They are best for photographs and detailed images.

Vector graphics are defined by mathematical equations for points, lines, and curves. They are infinitely scalable without losing quality. Common vector formats include SVG and AI. They are best for logos, icons, and illustrations.

Common File Formats

  • JPEG (Joint Photographic Experts Group): A compressed format best for photographs. It does not support transparency.
  • PNG (Portable Network Graphics): A lossless format that supports transparency. Great for web graphics and images with transparent backgrounds.
  • GIF (Graphics Interchange Format): Supports animation and a limited color palette. Good for simple animations.
  • SVG (Scalable Vector Graphics): A vector format that is great for logos and icons on the web.
  • PSD (Photoshop Document): The native file format for Adobe Photoshop, which preserves layers and other editing information.
