Topic 5: 2D Animation

Background

Frame-by-Frame vs. Motion Tweening

Frame-by-frame animation is a traditional animation technique where each frame is drawn individually. It allows for a high degree of control but can be very time-consuming.

Motion tweening is a technique where you create a starting and an ending keyframe, and the software automatically generates the frames in between. It is much faster than frame-by-frame animation but offers less control.

Digital Animation Tools

Tools like Krita, Adobe Animate, and After Effects provide powerful features for creating 2D animations, including layers, timelines, and onion skinning (which allows you to see previous and next frames as you draw).
